# Quick Installation Guide

## Prerequisites

✓ XAMPP installed
✓ Apache and MySQL running

## Installation Steps

### 1. Database Setup

1. Open phpMyAdmin: http://localhost/phpmyadmin
2. Click "Import" tab
3. Select file: `database/appointment_system.sql`
4. Click "Go"

### 2. Access Application

Open browser and go to: http://localhost/MassegeAppointmentSystem

### 3. Login
